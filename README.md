# Second Assignment - Hypothesis Testing

## Install Required Libraries

To install the required R libraries, run the following command in your R
console:

```r
source("install.R")
```

To install Pandoc on macOS, use Homebrew:

```shell
brew install pandoc
```

## Unzip datasource

To unzip the datasource, run the following command in your terminal:

```shell
unzip data.zip
```
